# Tietosuojakäytäntö

## Yleistä
Tämä sovellus ei kerää käyttäjätietoja eikä vaadi rekisteröitymistä. Käyttöäsi ei seurata eikä tallenneta henkilökohtaisia tietoja.

## Kerättävät tiedot
Sovellus ei kerää tai siirrä henkilötietoja, lokitietoja, analytiikkaa tai muita käyttäjätietoja.

## Rekisteröinti
Sovellus ei sisällä käyttäjärekisteröintiä.

## Kolmannen osapuolen palvelut
Sovellus ei käytä kolmannen osapuolen analytiikka- tai mainospalveluja, jotka keräisivät tietoja käyttäjistä.

## Tietojen säilytys ja poisto
Koska sovellus ei kerää tietoja, mitään käyttäjätietoja ei säilytetä eikä poistotoimia tarvita. Koska henkilötietoja ei käsitellä, sovellus ei luo rekisteriä eikä näin ollen sovelleta rekisterinpitäjän velvollisuuksia.

# Privacy Policy 

## Overview
This application does not collect any user data and does not require user registration. Your usage is not tracked and no personal information is stored.

## Information We Collect
The app collects no personal data, logs, analytics, or any other user information.

## Registration
The app contains no user registration feature.

## Third-Party Services
The app does not use third-party analytics or advertising services that collect user data.

## Data Retention and Deletion
As no data is collected, there is nothing to retain or delete. Because no personal data is processed, no user registry is created and registry-controller obligations do not apply.
